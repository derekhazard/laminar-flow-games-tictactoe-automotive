# Privacy Policy

**Laminar Flow Games Tic-Tac-Toe**
*Last updated: March 3, 2026*

## Overview

Laminar Flow Games Tic-Tac-Toe is an offline tic-tac-toe game designed for Android Automotive OS (AAOS). This app does not collect, store, transmit, or share any personal data whatsoever.

## Data Collection

This app collects **no data**. Specifically:

- **No personal information** is collected, stored, or transmitted
- **No analytics or tracking** of any kind
- **No advertising** or ad networks
- **No network access** — the app has no internet permission and makes no network requests
- **No accounts or sign-ins** are required or supported
- **No cookies or local storage** beyond the current session
- **No camera, microphone, location, or contacts** access

## In-Session Data

Game scores (wins, losses, draws) are tracked only during the current session for display purposes. This data exists solely in memory and is automatically discarded when the app is closed. No game data is written to disk or transmitted anywhere.

## Permissions

This app requests **no permissions**.

## Children's Privacy

This app does not collect any data from any user, including children.

If this privacy policy is updated, the revised version will be posted at this same URL with an updated date.
